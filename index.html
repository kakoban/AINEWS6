🌐 راهنمای به‌روزرسانی نسخه آنلاین (گیت‌هاب)

شما در حال حاضر در حال مشاهده نسخه آنلاین داشبورد هستید. از آنجا که داشبورد آنلاین استاتیک است، مرورگر امکان اجرای مستقیم فرآیند جمع‌آوری را ندارد. برای به‌روزرسانی امن و آنلاین اخبار، مراحل زیر را طی کنید:

  1. وارد حساب کاربری خود در GitHub و مخزن (Repository) این پروژه شوید.
  2. به تب Actions در بالای صفحه بروید.
  3. از منوی سمت چپ، گردش کار Daily AI Digest را انتخاب کنید.
  4. روی دکمه کشویی Run workflow کلیک کرده و دکمه سبز رنگ را فشار دهید.

سرورهای گیت‌هاب در پشت‌صحنه فعال شده و پس از حدود ۲ دقیقه، این صفحه به صورت خودکار با داده‌های تازه به‌روزرسانی خواهد شد. شما نیازی به ورود کلید API خود در مرورگر ندارید.
